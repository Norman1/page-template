About This Project

Welcome to the Learning Hub, a modern web application built with Web Components and designed to provide a clean, professional layout for educational content and tools.

Features

  • Responsive Holy Grail layout pattern
  • Google authentication integration
  • Dynamic navigation with user-specific content
  • Table of Contents generation
  • Modern Web Components architecture
  • No external dependencies

Technology Stack

This application is built using:

  • HTML5 Custom Elements (Web Components)
  • Vanilla JavaScript (ES6 modules)
  • CSS3 with modern features
  • Google Identity Services for authentication
