Getting Started Guide

Welcome to the Learning Hub framework. This guide will help you understand the core concepts and get your first project up and running quickly.

Prerequisites

Before you begin, ensure you have the following installed:

  • Modern web browser (Chrome 90+, Firefox 88+, Safari 14+)
  • Local web server (Python, Node.js, or VS Code Live Server)
  • Text editor (VS Code, WebStorm, or similar)
💡 Tip: This framework uses modern Web Components and ES6 modules, so a local server is required due to CORS restrictions.

Quick Setup

Get started in under 5 minutes with these simple steps:

1. Download the Template

Clone or download the Learning Hub template from the repository:

git clone https://github.com/your-repo/learning-hub.git
cd learning-hub

2. Start a Local Server

Choose one of these methods to serve your files:

Python: 
python -m http.server 8080
Node.js: 
npx http-server -p 8080
VS Code: 
Right-click index.html → "Open with Live Server"

3. Open in Browser

Navigate to http://localhost:8080 to see your site running.

✅ Success! You should now see the Learning Hub homepage with navigation, header, and content areas.

Core Concepts

Web Components Architecture

The framework is built on native Web Components, providing:

  • Encapsulation - Each component has isolated styles and behavior
  • Reusability - Components can be used across different pages
  • Native support - No framework dependencies required

Client-Side Routing

Navigation is handled by a lightweight router that:

  1. Listens for hash changes in the URL (#page1, #about)
  2. Loads the appropriate page component dynamically
  3. Updates the main content area without page refresh
⚠️ Note: Hash-based routing means URLs will include a # symbol. This is normal for client-side routing.

Responsive Layout

The framework uses a "Holy Grail" layout pattern that automatically adapts:

  • Desktop (1200px+): Three-column layout with sidebars
  • Tablet (768px-1199px): Collapsed navigation, main content prominent
  • Mobile (<768px): Single column with hamburger menu

Authentication Integration

The template includes optional Google authentication:

Setup Steps

  1. Create a Google Cloud project
  2. Enable the Google Identity API
  3. Configure OAuth 2.0 credentials
  4. Update config/google-auth.js with your client ID

Users can then sign in with their Google account, and their profile information will be saved locally.

Customization

Adding New Pages

To create a new page:

  1. Create an HTML template in templates/
  2. Create a page component in pages/
  3. Add the route to utils/router.js
  4. Add navigation link to components/nav-sidebar.js

Styling

The framework uses a dark theme by default. To customize:

  • Global styles: Edit styles/global.css
  • Component styles: Add CSS within each component's template
  • Color scheme: Update CSS custom properties in the root
